Виды подписок

Вы можете скачать приложение Spotify и слушать музыку бесплатно или подписаться на один из наших планов Spotify Premium.
    Бесплатная версия Spotify
  • Доступ более чем к 50 миллионам треков
  • Доступ к подкастам и аудиокнигам
  • Возможность слушать музыку за границей (не более 14 суток)
  • Воспроизведение треков по выбору на телефоне (только в определенных плейлистах)
    Spotify Premium:
  • Доступ более чем к 50 миллионам треков
  • Доступ к подкастам и аудиокнигам
  • Возможность слушать музыку за границей
  • Воспроизведение треков по выбору на телефоне
  • Без рекламы
  • Офлайн-режим
  • Очень высокое качество музыки

Индивидуальная подписка на Premium

Этот план предназначен для одного человека, который получает доступ ко всем перечисленным выше возможностям.

Premium для двоих

План Premium для двоих предназначен для двух человек, живущих вместе. Вы получаете: два аккаунта Spotify Premium по сниженной цене, "Микс для двоих" — общий плейлист, который подбирается с учетом музыкальных вкусов обоих пользователей.

Premium для семьи

План Premium для семьи предназначен для нескольких людей (до 6 человек), живущих вместе. Вы получате: до 6 аккаунтов Spotify Premium по сниженной цене, "Семейный микс" — общий плейлист, который подбирается с учетом музыкальных вкусов всех участников плана, Функция родительского контроля, позволяющая управлять доступом других пользователей к контенту для взрослых.
